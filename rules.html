PROTOCOLE DE JEU

1. MISE EN PLACE

  • Plateau de 5x5 cases au centre. Robots dans chaque coin.
  • Chaque joueur reçoit 3 points de vie.
  • Chaque joueur reçoit 8 tuiles de direction (1 de chaque sens).
  • Main de départ : Piochez 3 cartes événements + 1 par joueur manquant (4 joueurs étant le standard).
  • Le premier joueur est décidé au choix, puis sens des aiguilles d'une montre.

2. DÉROULÉ D'UN TOUR

PHASE 1 : CHOIX DES ÉVÉNEMENTS

Chaque joueur choisit 1 carte de sa main (+1 par joueur manquant/éliminé) et la place face cachée.

PHASE 2 : APPLICATION DES ÉVÉNEMENTS

Les cartes sont révélées. Les effets sont appliqués dans l'ordre du tour. Les événements globaux de fin de tour vont sur la piste dédiée.

PHASE 3 : CHOIX DES DÉPLACEMENTS

Chaque joueur place une tuile de direction face cachée devant lui.

PHASE 4 : APPLICATION DES DÉPLACEMENTS

Tous les joueurs effectuent leur déplacement en même temps.

PHASE 5 : ÉVÉNEMENTS IMMÉDIATS

Application des effets d'emplacements immédiats (Mines, Portails, etc.).

PHASE 6 : COMBAT

Si deux joueurs sont sur la même case :

  • Utilisation possible d'un jeton Armement pour gagner automatiquement avant le lancer.
  • Sinon, lancer de dé : le plus haut l'emporte.
  • Le vainqueur place le perdant sur une case adjacente libre.
  • Le perdant perd 1 vie.

PHASE 7 : ÉVÉNEMENTS DE FIN DE TOUR

Appliquez la piste d'événements (gauche à droite), puis les emplacements sur le terrain (Bleus avant Rouges).

PHASE 8 : FIN DU TOUR

Le jeton Premier Joueur tourne. Chaque joueur pioche jusqu'à avoir 3 cartes (+1 par joueur manquant/éliminé).

RÈGLES SUPPLÉMENTAIRES

  • Compteur de tours : Retournez les jetons Bombes à la fin de leur premier tour sur le plateau.
  • Réaction en chaîne : Si un emplacement rouge est détruit, son effet s'active immédiatement.
  • Éjection : Si une carte vous sort du plateau : -1 Vie, puis replacez votre robot sur la case de votre choix.
